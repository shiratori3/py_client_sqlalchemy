# 匹配规则

>
    头尾_任意匹配，中间_任意匹配（可扩充），第X项及倒数第Y项[]匹配，匹配项可以多于两个

    六位字符及以上：
        其他零星项目→
        +1：__[他零]_[星项]__
        -1： _[他零]_[星项]_
    五位字符：
        折旧摊销费 →
        +1：_[折旧]__[销费]_
        -1： [折旧]__[销费]
    四位字符：
        股份支付 →
        +1：_[股份]_[支付]_
        -1： [股份]_[支付]
    三位字符：
        单位一 → 
        +1：_[单位][位一]_
        -1： [单位][位一]
    两位字符：
        房租 → 
        +1：_[房租]_ 
        -1： [房租]
